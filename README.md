Тестовое задание
================

Простая инфраструктура, состоящая из:

- Nginx;
- Postgres;
- Node.js backend;
- Grafana;
- Prometheus + Exporters.

Функционал
----------

Проект поддерживает следующие функции:

- **Полный цикл CI/CD:** при получении обновленного кода начинается автоматический процесс сборки контейнеров и их пересылки на конечный сервер;
- **Мониторинг ресурсов и сети:** в проекте используются такие технологии, как *grafana* и *prometheus*, позволяющие в реальном времени получать информацию о состоянии сервера;
- **Мультиархитектурная сборка:** контейнеры можно развернуть как на x86_64 системах, так и на тех, что базируются на ARM архитектуре;
- **Изоляция сети:** выход в интернет имеют только *nginx* и *grafana*;
- **Ограничение ресурсов:** один сервис не может занять все ресурсы сервера.
- **Уведомления об окончании сборки:** после окончания сборки автоматически отправляется сообщение в Discord через webhook.

Инструкция по запуску
---------------------

Для запуска необходимо предварительно установить [certbot](https://github.com/certbot/certbot) для получения сертификатов. При использовании [Debian](https://debian.org/) или производного от него дистрибутива, загрузить его можно при помощи следующей команды:

```bash
apt install certbot
```

Для получения сертификатов воспользуйтесь следующей командой:

```bash
certbot certonly -d [DOMAINS]
```

Чтобы запустить проект, достаточно загрузить только `docker-compose.yml` файл и всю `.config` директорию в одну папку. Затем нужно перейти в нее и ввести:

```bash
docker compose up -d
```

После этого проект будет развернут автоматически.
